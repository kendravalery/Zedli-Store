# Zedli Store

Website e-commerce menggunakan Laravel-12, tailwind css, alpine.js

---

## Features
- Login
- Register
- Wishlist
- Cart
- Checkout
- address (Laravolt Indonesia)
- search (laravel scout)

---

## Tech Stack
- Laravel
- Tailwind CSS
- MySQL

---

## Folder Structure

resources/views/admin
→ halaman admin

---

## Cara Install

1. Clone project
2. composer install
3. npm install
4. php artisan migrate
5. php artisan serve
